# ⏱️ Timer

Timer simples desenvolvido com **HTML, CSS e JavaScript**, permitindo iniciar, pausar e zerar a contagem em tempo real no formato **HH:MM:SS**.

## 🚀 Acesse o projeto

🔗 **GitHub Pages:**
[https://LeandroSilvaN.github.io/timer/](https://LeandroSilvaN.github.io/timer/)

## 📌 Funcionalidades

* ▶️ Iniciar contagem
* ⏸️ Pausar timer
* 🔄 Zerar tempo
* 🕒 Atualização automática a cada segundo
* 🎨 Manipulação de classes para controle de estado

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (DOM e `setInterval`)
